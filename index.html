Extra flags for 4chan

Originally made for /int/, but we support /sp/, /pol/, and /bant/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the gitlab page

HOW TO INSTALL (Desktop/Mobile - Firefox/Chrome/Opera)

  1. Download Violentmonkey or Tampermonkey for Firefox, Tampermonkey for Chrome, or Violentmonkey for Opera.
  2. Install the Extra Flags script: Extra Flags for int.user.js. Otherwise for mobile users, install the Beta script: Extra Flags for 4chan-Mobile.user.js
  3. Browse to /int/
  4. A Tampermonkey page may appear. Click "Always allow domain"
  5. Click "OK" to set your region.
  6. Click "Always allow domain," a second time
  7. Set your region
  8. Reload the page
  9. Post from Quick Reply box (tap/click on a post number; top form does not work)
  10. To change regions, click the Violentmonkey/Tampermonkey menu->Extra Flags setup, and return back to the 4chan page to set your region from the popup menu
